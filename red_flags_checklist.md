# 🚩 Phishing Red Flag Checklist — Lohitaksh Raj

A quick reference for spotting phishing emails and messages.

## 1. Sender Red Flags
- Sender uses a free email (gmail/yahoo) but claims to be a company
- Domain has typos: `paypa1.com`, `amaz0n.com`, `microsft.com`
- Display name doesn't match the actual email address

## 2. Content Red Flags
- Generic greetings: "Dear Customer", "Dear User"
- Urgency: "within 24 hours", "act now", "immediate action"
- Threats: account closure, legal action, fund loss
- Too-good-to-be-true: lottery wins, free gift cards, prizes
- Poor grammar, spelling errors, weird capitalization

## 3. Link Red Flags
- URL shorteners (bit.ly, tinyurl) hiding real destinations
- Suspicious TLDs: `.xyz`, `.top`, `.tk`, `.ml`, `.click`
- Raw IP addresses instead of domain names
- Lookalike domains: `paypal.secure-login.xyz`
- `@` symbol inside URLs

## 4. Attachment Red Flags
- Unexpected `.zip`, `.exe`, `.scr`, `.js` files
- Office documents asking to "Enable Macros"

## 5. Request Red Flags
- Asks for passwords, OTPs, KYC, or banking details
- Asks for payment via gift cards, crypto, or wire transfer
- Asks you to keep it secret from others
